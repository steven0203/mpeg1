# C++ mpeg1 decoder
C++ mpeg1 decoder, use opencv4 only for display.
## Environment
C++  
opencv4  
Linux
## Compile
    make
## Run
    ./mpeg1_player <file_name>

## Test with some sample files
    ./mpeg1_player I_ONLY.M1V 
    ./mpeg1_player IP_ONLY.M1V 
    ./mpeg1_player IPB_ALL.M1V
